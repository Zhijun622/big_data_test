# 快速开始指南

## 第一步：安装依赖

```bash
cd 大数据导论/习题测验
npm install
```

## 第二步：启动应用

```bash
npm run dev
```

应用将在浏览器中自动打开，地址为：http://localhost:3000

## 第三步：使用系统

### 首次使用

1. 系统会自动从 `public/questions.json` 加载185道题目
2. 如果加载失败，可以手动导入：
   - 进入"题库管理"页面
   - 点击"导入JSON"按钮
   - 选择 `questions.json` 文件

### 开始练习

1. 点击主页的"开始测验"卡片
2. 选择测验模式：
   - **全部题目测验**：练习所有185道题目
   - **按单元测验**：选择特定章节（如"第1章 大数据概述"）
   - **随机抽取测验**：随机抽取20道题目（可自定义数量）
   - **错题专项测验**：只练习错题本中的题目
3. 开始答题，完成后查看结果和解析

### 查看学习进度

- **错题本**：查看所有答错的题目，可以标记为已掌握
- **学习统计**：查看总体正确率、各单元掌握情况、历史记录等

✅ 支持单选题和多选题  
✅ 自动收集错题  
✅ 详细的答题解析  
✅ 多种测验模式  
✅ 学习进度统计

- 所有数据保存在浏览器本地，清除浏览器数据会丢失记录
- 支持导入/导出JSON格式的题目文件
- 建议定期备份学习数据
