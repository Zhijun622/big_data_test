# 大数据导论选择题练习系统

一个功能完整、界面美观的大数据导论课程选择题练习网站，支持题库管理、智能测验和错题强化练习。

## 功能特性

### 1. 题库管理模块
- ✅ 支持按单元（章节）批量导入题目
- ✅ 支持单选题和多选题
- ✅ 题目列表展示，支持按单元筛选和搜索
- ✅ 支持JSON格式导入题目

### 2. 测验功能模块
- ✅ 全部题目测验
- ✅ 按单元测验
- ✅ 随机抽取测验（指定数量）
- ✅ 错题专项测验
- ✅ 答题界面：显示题目序号、进度条、上一题/下一题导航
- ✅ 实时反馈：提交后立即显示对错、正确答案和解析

### 3. 成绩记录与分析模块
- ✅ 自动保存每次测验结果
- ✅ 记录时间、得分、正确率
- ✅ 总体正确率统计
- ✅ 各单元掌握情况（柱状图）
- ✅ 历史测验记录时间线
- ✅ 正确率趋势图（折线图）

### 4. 错题本功能
- ✅ 自动收集所有答错的题目
- ✅ 显示错误次数
- ✅ 支持一键开始错题测验
- ✅ 答对后可标记为"已掌握"

### 5. 数据持久化
- ✅ 使用 window.storage API 保存数据
- ✅ 所有数据自动保存到本地

## 技术栈

- **React 18+** - 使用函数组件和Hooks
- **React Router** - 路由管理
- **Tailwind CSS** - 样式方案
- **lucide-react** - 图标库
- **recharts** - 数据可视化（统计图表）
- **Vite** - 构建工具

## 安装和运行

### 1. 安装依赖

```bash
cd 大数据导论/习题测验
npm install
```

### 2. 解析题目（可选）

如果还没有生成 `questions.json` 文件，可以运行：

```bash
python3 parse_questions.py
```

这会将所有章节的markdown文件解析为JSON格式的题目数据。

### 3. 启动开发服务器

```bash
npm run dev
```

应用将在 http://localhost:3000 启动

### 4. 构建生产版本

```bash
npm run build
```

## 使用说明

### 首次使用

1. 进入"题库管理"页面
2. 如果题库为空，系统会自动尝试加载 `questions.json` 文件
3. 也可以手动导入JSON格式的题目文件

### 开始测验

1. 点击"开始测验"
2. 选择测验模式：
   - **全部题目测验**：测验所有题目
   - **按单元测验**：选择特定章节
   - **随机抽取测验**：随机抽取指定数量题目
   - **错题专项测验**：只测验错题本中的题目
3. 开始答题，可以随时切换上一题/下一题
4. 提交后查看结果和解析

### 查看统计

- 在"学习统计"页面查看：
  - 总体数据（总题数、已练习题数、总正确率、错题总数）
  - 练习进度饼图
  - 正确率趋势图
  - 各单元掌握情况柱状图
  - 历史测验记录

### 错题本

- 所有答错的题目会自动添加到错题本
- 可以查看错题详情和解析
- 可以标记错题为"已掌握"
- 可以一键开始错题专项练习

## 题目数据格式

题目JSON格式示例：

```json
{
  "questions": [
    {
      "id": "q_1_1_single",
      "unit": "第1章 大数据概述",
      "type": "single",
      "question": "下面关于数据的说法，错误的是",
      "options": [
        { "key": "A", "text": "选项A内容" },
        { "key": "B", "text": "选项B内容" },
        { "key": "C", "text": "选项C内容" },
        { "key": "D", "text": "选项D内容" }
      ],
      "correctAnswer": ["B"],
      "explanation": "解析内容...",
      "difficulty": "medium"
    }
  ]
}
```

## 注意事项

1. 所有数据使用 `window.storage` API 保存，如果该API不可用，会自动降级到 `localStorage`
2. 题目数据保存在本地，清除浏览器数据会丢失所有记录
3. 支持导入JSON格式的题目文件进行批量导入
