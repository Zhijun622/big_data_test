# 错题回顾与分析工具包

本工具包帮助你从大数据导论习题测验系统中提取错题，进行系统分析，并生成详细的学习报告。

## 📦 包含文件

```
大数据导论/
└── 习题测验/                      # 测验系统
    ├── src/
    │   ├── pages/
    │   │   ├── WrongQuestions.jsx # 错题本页面
    │   │   └── Statistics.jsx     # 统计页面
    │   └── utils/
    │       └── dataManager.js     # 数据管理
    ├── questions.json             # 题库数据
    └── 错题/                      # 错题分析工具包（当前目录）
        ├── README.md                     # 本文件
        ├── extract_quiz_data.js          # 浏览器端数据提取脚本
        ├── 错题分析使用指南.md            # 详细使用说明
        ├── 快速操作指南.md                # 5分钟快速上手
        └── 错题分析报告模板.md            # 标准报告模板
```

## 🚀 快速开始

### 最简单方法（推荐新手）

1. **打开测验系统**
   - 进入 `习题测验` 目录
   - 在浏览器中打开系统

2. **查看错题本**
   - 点击导航栏"错题本"
   - 点击"从历史记录恢复错题"
   - 查看所有错题和解析

3. **开始练习**
   - 点击"开始练习"按钮
   - 进行错题专项训练

### 高级方法（生成详细报告）

1. **打开浏览器控制台**
   - 按 F12 或 Ctrl+Shift+I (Windows/Linux)
   - 按 Cmd+Option+I (Mac)

2. **运行提取脚本**
   ```javascript
   // 复制 extract_quiz_data.js 的全部内容到控制台
   // 然后执行：
   const result = main();
   ```

3. **保存报告**
   - 复制控制台输出的Markdown报告
   - 保存为文件，如：`我的错题分析_2024-12-28.md`

## 🔧 系统功能说明

### 错题本功能
- **查看错题**：显示所有做错的题目
- **章节筛选**：按章节过滤错题
- **错误统计**：显示每道题的错误次数
- **开始练习**：进入错题专项训练模式
- **标记掌握**：标记已掌握的题目
- **恢复错题**：从历史记录重建错题本

### 统计功能
- **总体统计**：题库总数、已练习数、总正确率
- **进度可视化**：饼图显示练习进度
- **趋势分析**：折线图显示正确率趋势
- **章节掌握**：柱状图显示各章节情况
- **历史记录**：表格显示所有测验记录

### 数据导出功能
- **完整数据**：题库、记录、错题
- **智能分析**：自动分析错误原因
- **报告生成**：Markdown格式报告
- **离线查看**：可打印或保存为PDF

## ⚠️ 注意事项

### 数据安全
- ✓ 数据存储在浏览器localStorage
- ✓ 不会上传到服务器
- ✓ 清除浏览器数据会丢失记录
- ✓ 建议定期导出备份

### 浏览器兼容
- ✓ Chrome/Edge（推荐）
- ✓ Firefox
- ✓ Safari
- ⚠ 需要启用JavaScript和localStorage

### 使用建议
- ✓ 同一浏览器同一账号使用
- ✓ 不要在隐私模式下使用
- ✓ 定期导出数据备份
- ✓ 考前一定要备份数据

## 🆘 遇到问题？

### 常见问题速查

| 问题 | 解决方案 |
|------|----------|
| 错题本是空的 | 点击"从历史记录恢复错题" |
| 数据丢失了 | 检查是否清除了浏览器数据 |
| 导出报告失败 | 检查控制台错误信息 |
| 题目显示不全 | 刷新页面或清除缓存 |
