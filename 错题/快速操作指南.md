## 📝 具体操作步骤

### 步骤1：访问错题本（1分钟）

1. 打开习题测验系统
2. 在顶部导航栏点击"错题本"
3. 查看错题数量

**如果显示"暂无错题"：**
- 点击"从历史记录恢复错题"按钮
- 系统会自动从所有测验记录中提取错题
- 刷新页面即可看到错题

### 步骤2：筛选和查看（2分钟）

**按章节筛选：**
- 在下拉菜单中选择要查看的章节
- 或选择"全部章节"查看所有错题

**查看题目详情：**
- 每道错题显示：
  - 错误次数
  - 章节名称
  - 题型（单选/多选）
  - 题目内容
  - 所有选项
  - 正确答案（绿色标注）
  - 详细解析

### 步骤3：开始练习（建议10-30分钟）

**练习方式：**
1. 点击右上角"开始练习"按钮
2. 进入错题专项练习模式
3. 逐题作答，查看解析
4. 完成后查看成绩

**练习技巧：**
- 认真阅读题目，注意关键词
- 做题前先回忆知识点
- 做错后仔细看解析
- 理解原理而非死记答案

### 步骤4：标记已掌握（持续进行）

**什么时候标记已掌握？**
- 连续3次做对该题
- 完全理解题目考点
- 能够解释给别人听

**如何标记：**
- 点击题目右上角"标记已掌握"按钮
- 已掌握的题目不会再出现在练习列表中
- 但仍然保留在错题本中

### 步骤5：查看学习统计（2分钟）

1. 点击导航栏"统计"
2. 查看：
   - 总题数、已练习题数、总正确率
   - 各章节掌握情况（柱状图）
   - 正确率趋势（折线图）
   - 历史测验记录表格

**重点关注：**
- 哪些章节正确率低？→ 需要重点复习
- 正确率是否在提升？→ 学习是否有效
- 最近几次测验表现？→ 当前状态

## 💡 高效使用技巧

### 技巧1：定期导出报告
```
频率：每周一次
目的：跟踪学习进度，发现薄弱环节
操作：使用extract_quiz_data.js导出
```

### 技巧2：分类复习
```
第一轮：只看高频错题（错误≥3次）
第二轮：复习中频错题（错误2次）
第三轮：全面复习所有错题
```

### 技巧3：建立错题笔记
```
对于每道高频错题：
1. 记录题目关键信息
2. 总结错误原因
3. 整理相关知识点
4. 记录避免错误的方法
```

### 技巧4：定时复习
```
建议复习周期：
- 第1天：首次做错
- 第2天：第一次复习
- 第4天：第二次复习
- 第7天：第三次复习
- 第15天：第四次复习
```

### 技巧5：结合章节复习
```
步骤：
1. 查看统计页面，找出薄弱章节
2. 重新学习该章节知识点
3. 重做该章节的错题
4. 做新的该章节练习题
5. 对比前后正确率
```

## ⚠️ 常见问题解决

### Q1: 点击"从历史记录恢复错题"没反应？
```
解决方法：
1. 检查浏览器控制台是否有错误
2. 确认是否有历史测验记录
3. 尝试刷新页面
4. 清除浏览器缓存后重试
```

### Q2: 错题数量不对？
```
可能原因：
- 某些测验记录没有保存成功
- 浏览器localStorage被清理
- 数据同步问题

解决方法：
1. 查看统计页面的历史记录数
2. 重新从历史记录恢复错题
3. 如果数据丢失，重新做一遍测验
```

### Q3: 如何查看具体某次测验的错题？
```
方法1：统计页面
- 查看历史测验记录表格
- 记下该次测验的时间和正确率

方法2：导出报告
- 使用脚本导出完整数据
- 在records数组中找到对应的记录
- 查看details字段中isCorrect为false的题目
```

### Q4: 如何重置所有数据重新开始？
```
警告：此操作不可恢复！

方法：在浏览器控制台执行
localStorage.clear()
location.reload()
```

## 📊 数据备份建议

### 定期备份
```javascript
// 在浏览器控制台运行
const backup = {
  date: new Date().toISOString(),
  records: localStorage.getItem('quiz-records'),
  wrong: localStorage.getItem('wrong-questions'),
  questions: localStorage.getItem('quiz-bank')
};
console.log(JSON.stringify(backup));
// 复制输出内容保存到文本文件
```

### 恢复备份
```javascript
// 在浏览器控制台运行
const backup = {/* 粘贴备份的JSON数据 */};
localStorage.setItem('quiz-records', backup.records);
localStorage.setItem('wrong-questions', backup.wrong);
localStorage.setItem('quiz-bank', backup.questions);
location.reload();
```
